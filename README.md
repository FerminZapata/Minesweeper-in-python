English
You can change the difficulty by changing the variable "mode"
There are three modes you can play: "easy", "normal", "hard"
If you write a difficulty not registered, then mode will be set in easy mode
When you lose you'll need to close and open the file (Sorry about that! ^_^')
This isn't like the google minesweeper, so you can lose on your first move or start in a numbered tile without being able to see any blank tiles

Spanish
Podes cambiar la dificultad cambiando la variable "mode"
Hay tres modos en los que podes jugar: "easy", "normal", "hard"
Si escribis una dificultad que no fue definida, el modo por defecto va a ser facil
Cuando perdes tenes que cerrar y abrir el archivo de vuelta (Perdon! ^_^')
Este no es como el buscaminas de google, entonces podes perder en tu primer movimiento o comenzar en una casilla con numero sin ser capaz de ver las blancas
